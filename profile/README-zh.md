# ESCSS - 逃離開發者的噩夢
![logo](https://github.com/ESCSS-labs/ESCSS/blob/main/assets/logo.png)

## 特色
  - **安全性**: 開源專案確保透明度，且 ESTest 沒有任何依賴項。
  - **開箱即用**: 靈活的 JavaScript 和 CSS 工具，能與 Legacy codebase 無縫整合使用。
  - **100% 輸入覆蓋率**: 全面的輸入測試覆蓋率能最大限度減少錯誤，增強客戶的信任。
  - **錯誤捕獲**: 還在花費數小時通靈般地尋找錯誤嗎？數秒內即可定位錯誤來源。
  - **開發者體驗**: 專為開發者設計的工具，提升產品品質和開發者的成就感，並幫助企業留住人才。
